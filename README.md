
# Customized Latex subfigure package

Original subfigure package arranges subfigures (subtables) and subfigurecaptions (subtablecaptions) vertically. This version is customized to put subcaptions alongside subfigures (subtables).

## Install

    make
    cp subfigure.sty <document path>

## Usage

The usage is the same as original subfigure package. Use `\subfiguretopcaptrue` (`\subfiguretopcapfalse`) to put subfigure (subtable) captions at the left (right) side of figures (tables).
